Project Overview

This project is designed to develop web programming and interaction design skills. My situation is I am an employee in a fictitious company, tasked with making an interactive 3D and audio ‘browser experience’ shared on social media channels to raise the company’s profile online. To complete this project, Three.js and Tone.js are used to design and make an interactive piece that will promote play and creativity as well as reinforce the company’s brand identity.

Tools Used

  • JavaScript
  • HTML
  • CodeSandbox
  • Three.js
  • GLTF model
